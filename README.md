# Task Manager

## Overview
Task Manager is a responsive task management application built using React and Tailwind CSS. It allows users to add, filter, and manage tasks efficiently with a dark mode toggle.

## Features
- Add, delete, and toggle task status (completed/incomplete)
- Filter tasks by status (All, Active, Completed)
- Search tasks by name or priority
- Priority-based task highlighting
- Dark mode support
- Responsive design for all screen sizes

## Technologies Used
- Vite
- React
- TypeScript
- Custom Hook
- Tailwind CSS
- Lucide Icons


## Installation

1. Clone the repository:
   ```sh
   git clone "https://github.com/shubhambendkhale77/Task_Manager.git"
   ```
2. Navigate to the project directory:
   ```sh
   cd Task_Manager
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```

## Usage
1. Enter a task in the input field and select a priority.
2. Click the "Add Task" button to add it to the list.
3. Use the search bar to find tasks by name or priority.
4. Filter tasks using the dropdown menu.
5. Click on a task to mark it as completed/incomplete.
6. Toggle dark mode for a better viewing experience.
